# SimpleRAG

A simple NotebookLM-style RAG application built using:

- FastAPI
- ChromaDB
- HuggingFace Embeddings
- OpenRouter LLMs

## Features

- Upload PDF
- Chunking
- Vector Embeddings
- Semantic Retrieval
- Grounded AI Answers

## Tech Stack

- FastAPI
- ChromaDB
- HuggingFace Sentence Transformers
- OpenRouter
- HTML/CSS/JavaScript

## Run Backend

```bash
uvicorn main:app --reload
```

## Run Frontend

Use VS Code Live Server.
